# organismIDSource

## Definition 
URN denoting the globally unique identifier as assigned by the repository publishing the dataset. 

## Format
urn:catalog:[repository]:[institution]:[projectcode]:[OrganismID], e.g. urn:catalog:otn:Dalhousie:NSBS:Brandy

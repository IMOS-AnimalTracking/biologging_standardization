# sensorCalibrationDate

## Definition 
Date when sensor calibration was done.

## Standard
ISO 8601

## Format
Datetime in UTC, yyyy-MM-ddT HH:mm:ss.SSSZ, e.g. “2020-03-29T 17:56:10.000Z”
